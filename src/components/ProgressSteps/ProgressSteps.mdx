## Examples

### Sizes

Use `size` to control the scale of step indicators.

<Canvas of={ProgressStepsStories.Sizes} sourceState='hidden' />
<div className="canvas-controls-only">
<Controls of={ProgressStepsStories.Sizes}/>
</div>

### Vertical

Use `orientation="vertical"` to display steps in a vertical layout, ideal for sidebar navigation or multi-step forms.

<Canvas of={ProgressStepsStories.Vertical} />

### With Icons

Use `icon` prop on `ProgressSteps.Step` to display custom icons instead of step numbers.

<Canvas of={ProgressStepsStories.WithIcons} />

### Interactive

Example with navigation controls.

<Canvas of={ProgressStepsStories.Interactive} />

### All States

Shows all step states: completed, active, and inactive.

<Canvas of={ProgressStepsStories.AllStates} />

### Dashed Connector

Use `connectorStyle="dashed"` for a dotted line between steps.

<Canvas of={ProgressStepsStories.DashedConnector} sourceState='hidden' />
<div className="canvas-controls-only">
<Controls of={ProgressStepsStories.DashedConnector}/>
</div>

### Minimal

Use `variant="minimal"` for a compact horizontal bar indicator. Requires `total` prop.

<Canvas of={ProgressStepsStories.Minimal} />

### Colors

Use `color` to change the color scheme of completed and active steps.

<Canvas of={ProgressStepsStories.SuccessColor} sourceState='hidden' />
