## Examples

### Avatar

Use a circular skeleton to represent loading avatars. The default size matches Plex UI's avatar component (28px).

<CollapsibleCanvas of={SkeletonStories.Avatar} />

### Card

Compose multiple skeletons to represent loading cards with images and text content.

Multiple text lines with varying widths create natural-looking paragraph placeholders.
