## Examples

### Different triggers
`Tooltip` provides an optional component, `Tooltip.TriggerDecorator`, to help create accessible tooltip triggers. However, any component can be used as a trigger.
<CollapsibleCanvas of={TooltipStories.Triggers} />
<CollapsibleSource of={TooltipStories.Triggers} />

### Conditionally enabled
Sometimes a tooltip should only be disabled under certain conditions, such as when a component is disabled, providing more context as to why that is. Below is an example of how to do this by conditionally passing in `null` content.
<CollapsibleCanvas of={TooltipStories.Conditional} />
<div className="canvas-controls-only">
<Controls of={TooltipStories.Conditional} />
</div>
<CollapsibleSource code={`<Tooltip
  content={disabled ? "This action is disabled for reasons" : null}
>
  <Button color="primary" size="lg" disabled={disabled}>
    Sample button
  </Button>
</Tooltip>
`} />

### Compact tooltips
Compact tooltips are designed for short, 1-3 word tooltips with inverted styling. They feature minimal padding, a dark background with white text, and no box shadows. Use `compact={true}` for brief status messages.
<CollapsibleCanvas of={TooltipStories.Compact} />
<CollapsibleSource code={`<Tooltip content="Copy to clipboard" compact>
  <Button color="secondary" variant="ghost" size="md">
    <Copy />
  </Button>
</Tooltip>
`} />
