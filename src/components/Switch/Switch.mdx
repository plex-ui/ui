## Examples

### With label
You can add a label to the component by using the `label` prop. The label will automatically connect to the switch with `id`.
<CollapsibleCanvas of={SwitchStories.WithLabel} />
<CollapsibleSource of={SwitchStories.WithLabel} />

### Label position
Set `labelPosition="start"` to place the label on the left side of the switch control.
<CollapsibleCanvas of={SwitchStories.LabelPositionStart} />
<CollapsibleSource of={SwitchStories.LabelPositionStart} />

### Default checked
Initialize the switch with `defaultChecked` when you don't need to control its state. You can then use the `name` prop to read the value later on.
<CollapsibleCanvas of={SwitchStories.DefaultChecked} />
<CollapsibleSource of={SwitchStories.DefaultChecked} />

### Controlled
Pass `checked` and `onCheckedChange` to manage the state in a controlled manner.
<CollapsibleCanvas of={SwitchStories.Controlled} />
<CollapsibleSource of={SwitchStories.Controlled} />

### Disabled
Use `disabled` to prevent the user from toggling the control.
<CollapsibleCanvas of={SwitchStories.Disabled} />
<CollapsibleSource of={SwitchStories.Disabled} />
